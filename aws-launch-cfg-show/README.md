# SYNOPSIS

Sparrow plugin to print launch configuration in human readable format.


# INSTALL

```
s6 --install aws-launch-cfg-show
```

# USAGE

Dump in human readable format:

```
s6 --plg-run aws-launch-cfg-show@name=bla_bla_bla
```
