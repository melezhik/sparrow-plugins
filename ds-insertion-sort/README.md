# ds-insertion-sort

Insertion sort. Raku and Python implementation

# Install

    s6 --install ds-insertion-sort

# Usage

Raku:

    s6 --plg-run ds-insertion-sort

Python:

    s6 --plg-run ds-insertion-sort@language=python
