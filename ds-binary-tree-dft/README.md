# ds-binary-tree-dft

An example of binary tree depth first traversal using Raku

# Install

    s6 --install ds-binary-tree-dft

# Usage

    s6 --plg-run ds-binary-tree-dft

# Other languages

    s6 --plg-run ds-binary-tree-dft@language=python
