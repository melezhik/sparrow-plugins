# ds-graph-bft

Directed graph breadth first traversal. Raku and Python implementation.

# Install

    s6 --install ds-graph-bft

# Usage

Raku:

    s6 --plg-run ds-graph-bft

Python:

    s6 --plg-run ds-graph-bft@language=python

https://www.geeksforgeeks.org/graph-bft/
