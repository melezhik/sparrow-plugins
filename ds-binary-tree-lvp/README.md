# ds-binary-tree-lvp

Print left view of binary tree using Raku.

2 methods ( interative with FIFO and recursive ) are used.

# Install

    s6 --install ds-binary-tree-lvp

# Usage

    s6 --plg-run ds-binary-tree-lvp
