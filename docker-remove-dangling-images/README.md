# SYNOPSIS

Remove dangling docker images and stopped containers.

# INSTALL

    $ s6 --install docker-remove-dangling-images

# USAGE

    $ sparrow plg run docker-remove-dangling-images

# Platforms supported

* Linux

* Windows
