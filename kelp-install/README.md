# SYNOPSIS

Installs [kelp](ttps://github.com/crhuber/kelp) package manager

# INSTALL

    s6 --install kelp-install

# USAGE

    s6 --plg-run kelp-install

# Parameters


## with-sudo

Use sudo when install

```bash
s6 --plg-run kelp-install@with-sudo
```
