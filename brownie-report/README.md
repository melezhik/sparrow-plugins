# SYNOPSIS

Utility to generate brownie report


# INSTALL

    $ s6 --install brownie-report

# USAGE

Cli

    # with default settings    
    $ s6 --plg-run brownie-report@new=sha1,old=sha2

# Parameters

* `new`

Rakudo version commit

* `old` 

Rakudo version commit
