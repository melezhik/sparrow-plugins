# SYNOPSIS

Hello world Bash plugin, see https://github.com/melezhik/Sparrow6/blob/master/posts/CliAppDevelopement.md

# INSTALL

    s6 --install hello

# USAGE

    s6 --plg-run hello@who=Bash,ok

# Plugin parameters

## who

String to echo out

## ok

Boolean parameter
