# build-essential

Installs build-essential meta package

# Install

    s6 --install build-essential

# Usage

    s6 --plg-run build-essential

## Parameters

## sudo

Use sudo when install

```bash
s6 --plg-run build-essential@sudo
```

# Platforms supported

* Ubuntu

* Debian

* Alpine
